使用方法:

`bvf | bvinfo + b23tv短链网址 | bilibili.com网址 | Bv号`

示例:

+ .bvf BV1QD4y1b7BL
+ .bvf https://www.bilibili.com/video/BV1ZD4y1678s/?spm_id_from=333.337.search-card.all.click&vd_source=e3eecd37061be8a85e5e4844ee60580a
